# Contributing to AI Agency Starter Kit

First off, thank you for considering contributing to the AI Agency Starter Kit! It's people like you that make this open-source project such a great tool for the community.

## 🤝 How You Can Contribute

We welcome contributions in many forms:

1. **New Templates**: Add contracts, proposals, or scripts for new niches.
2. **Prompt Engineering**: Add new, tested chatbot or voice-agent prompts.
3. **Interactive Tools**: Improve the CSS/JS of existing calculators or add new ones.
4. **Bug Fixes**: Fix typos, broken links, or formatting issues in the Markdown files.
5. **Translations**: Translate our core documents into other languages.

## 📝 Submission Guidelines

### 1. Quality Standards
* **No Placeholder Text**: Do not use `lorem ipsum`. Provide realistic, actionable examples.
* **Clear Customization Points**: If a user needs to input their own data, use clear bracketed text like `[Agency Name]` or `[Client Industry]`.
* **Professional Tone**: All documents should reflect a high standard of B2B professionalism.

### 2. Pull Request Process
1. Fork the repository and create your branch from `main`.
2. If you've added code (like a new calculator), ensure it is well-commented and works across major browsers.
3. If you've added a template, ensure the formatting is clean GitHub Flavored Markdown.
4. Update the `README.md` if your contribution adds a major new feature or folder.
5. Issue that pull request!

### 3. Adding to Interactive Tools
If you are modifying or adding a tool in the `/tools` directory:
* Keep it dependency-free (Vanilla HTML/CSS/JS only) to ensure it's easy for non-technical users to host.
* Ensure the design is responsive.
* Include a `README.md` in the tool's folder with usage instructions and screenshot placeholders.

## 🐛 Reporting Bugs

If you find a bug (e.g., a broken calculator script or a glaring typo), please open an issue and include:
* A descriptive title.
* Steps to reproduce the issue (if applicable).
* What you expected to happen.

Thank you for helping us build the best resource for AI automation agencies!
