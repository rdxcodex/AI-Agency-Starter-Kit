# Top AI APIs for Developers

**Purpose:** The core foundational models and APIs you need to power your automated workflows and chatbots.

## 1. OpenAI (ChatGPT)
* **Best For:** General purpose text generation, system prompts, and logic routing.
* **Key Models:** `gpt-4o` (fast, multimodal, best for chatbots), `gpt-4-turbo` (deep reasoning).
* **Agency Use Case:** Powering the brain of your web chatbots or drafting personalized cold emails inside Make.com.

## 2. Anthropic (Claude)
* **Best For:** Long-form writing, coding, and parsing massive documents.
* **Key Models:** `Claude 3.5 Sonnet` (incredible speed/intelligence ratio), `Claude 3 Opus`.
* **Agency Use Case:** If a client gives you a 200-page PDF manual and wants a bot to answer questions based *strictly* on that manual, Claude is often better at following strict instructions and avoiding hallucinations than GPT-4.

## 3. Perplexity API
* **Best For:** Real-time web search and research automation.
* **Agency Use Case:** In a lead generation workflow, you can send a company URL to Perplexity via API and ask it to return a 3-sentence summary of recent company news to use in your cold email.

## 4. ElevenLabs
* **Best For:** Text-to-Speech (TTS) and voice cloning.
* **Agency Use Case:** Generating the hyper-realistic audio for your Voice Agents or creating automated personalized voicemail drops.

## 5. Groq
* **Best For:** Ultra-low latency inference (speed).
* **Agency Use Case:** When building voice agents from scratch, LLM response time is the biggest bottleneck. Groq runs open-source models (like Llama 3) at blistering speeds, making voice conversations feel instantaneous.
