# AI Agency Starter Kit - Roadmap

This roadmap outlines the future vision and planned features for the AI Agency Starter Kit. We are constantly looking to improve and expand this repository to provide maximum value to agency owners.

## 📌 Q3 2026 Goals
- [ ] **Expanded Niches**: Add dedicated proposal and prompt templates for Healthcare, Legal, and Logistics niches.
- [ ] **Advanced Calculators**: Develop a "Chatbot Token Cost Estimator" tool.
- [ ] **SOP Library**: Introduce a new `/sops` directory containing standard operating procedures for hiring virtual assistants and developers.
- [ ] **Video Tutorials**: Link to community-created video walk-throughs on how to deploy the interactive tools.

## 📌 Q4 2026 Goals
- [ ] **Make.com / Zapier Blueprints**: Provide downloadable JSON blueprints for common automation workflows.
- [ ] **Cold Email Sequences**: Expand the sales scripts to include 6-month automated nurturing sequences.
- [ ] **Figma Templates**: Add open-source Figma files for client dashboards and proposal designs.

## 📌 2027 and Beyond
- [ ] **Agency OS Dashboard**: A lightweight, open-source Next.js dashboard template to manage clients, built on top of this repository's frameworks.
- [ ] **Multi-language Support**: Translate the entire starter kit into Spanish, French, and German.
