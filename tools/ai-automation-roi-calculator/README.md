# AI Automation ROI Calculator

This interactive calculator helps agencies clearly demonstrate the financial value of their automation solutions to prospective clients. By inputting the client's current manual processes and the proposed AI solution, the calculator outputs hard numbers on monthly savings, annual savings, and total ROI.

## 🚀 Features

* **Dynamic Calculation:** Instantly calculates monthly/annual savings and ROI percentage.
* **Modern UI:** Clean, trustworthy design using standard B2B colors (blues and greens).
* **Responsive:** Fully functional on mobile for on-the-go sales pitches.
* **Financial Formatting:** Automatically formats outputs into readable currency strings.

## 🛠️ Usage Instructions

1. Open `index.html` in your browser to use the tool locally.
2. To embed on a WordPress, Webflow, or custom site, copy the HTML into an embed block and link the provided `style.css` and `script.js`.
3. Use this tool during live discovery calls to build a compelling business case for the client.

## 📝 Example Inputs & Outputs

**Example Input:**
* Hours Spent Per Week: 20
* Average Hourly Wage: $25
* Monthly Cost of Automation: $500
* Estimated Efficiency Gain: 80%

**Example Output:**
* **Monthly Savings:** $1,232
* **Annual Savings:** $14,784
* **Hours Saved Annually:** 831.4 hrs
* **ROI:** 246.4%
