# Website Audit Score Calculator

This is a fully functional interactive tool designed to help AI Automation Agencies grade a prospect's website and provide actionable feedback. It serves as an excellent lead magnet or a visual aid during sales discovery calls.

## 🚀 Features

* **Clean UI:** Built with modern CSS variables and flexbox.
* **Mobile Responsive:** Looks great on mobile devices and desktops alike.
* **Input Validation:** Prevents scores that exceed the maximum limit for each category.
* **Dynamic Insights:** Generates different sales pitches based on the final calculated score.
* **Score Animation:** Smooth count-up animation for a premium feel.

## 🛠️ Usage Instructions

This tool requires zero dependencies or build steps. 

1. Simply open `index.html` in your web browser.
2. To embed this on your agency website, copy the HTML into your page structure and link the `style.css` and `script.js` files.
3. You can customize the scoring thresholds and pitch text inside `script.js`.

## 🤝 For Contributors

If you want to contribute to this tool:
1. Ensure you stick to vanilla HTML, CSS, and JavaScript. We do not want to introduce frameworks (like React or Vue) here to ensure maximum compatibility for non-technical users.
2. If you add new criteria, ensure the total score always sums to 100.
3. Keep the UI design aligned with the other tools in this repository.
